Technical Guide

Drill Hole Planning with Diomine DH Auditor 2026.1.2

Rigorous drill hole planning is the cornerstone of any successful mineral exploration campaign. With the release of Diomine DH Auditor version 2026.1.2, geologists now have access to advanced tools designed to optimize trajectories and reduce operational costs.

Key Stages of the Technical Workflow

1. Project Initialization

Import digital terrain models (DTM) and 3D block models to visualize mineralized zones in their spatial context.

2. Target Definition

Precisely position target intersection points based on interpreted geological structures.

3. Trajectory Calculation

The software automatically generates the optimal azimuth and dip from the surface collar to the specified target.

4. Deviation Management

Anticipate path variations based on historical data to prevent missing high-grade targets.

Why Version 2026.1.2 is a Game Changer

Unlike manual plotting methods, this update integrates real-time data validation. If a planned trajectory intersects an existing underground void or legacy gallery, the system alerts the user immediately. Furthermore, exporting data to drilling contractors is now seamless via standardized CSV or DXF formats.

Utilizing Diomine DH Auditor secures your exploration investment by ensuring that every meter drilled provides high-value geological information.
